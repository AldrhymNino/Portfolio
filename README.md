# Portfolio Aldrin - React + Vite + Tailwind

Proyecto base minimalista para portafolio personal (React + Vite + Tailwind + Framer Motion).

## Pasos para usar

1. Instalar dependencias:
   ```
   npm install
   ```

2. Iniciar en modo desarrollo:
   ```
   npm run dev
   ```

3. Construir para producción:
   ```
   npm run build
   npm run preview
   ```
